## 一、产品背景与定位

### 1.1 背景
用户自建 AI 陪伴网页应用「我的手机」，目前已部署至 GitHub Pages，具备聊天、记账、经期、朋友圈、计时器等基础功能。当前为单文件 SPA 架构（index.html 约 203KB），代码与样式全部耦合。

### 1.2 产品定位
一款**以角色陪伴为核心的桌面级互动工具**，模拟真实手机界面，用户可与自定义 AI 角色（如原创梦角「骆云影」）进行沉浸式交流、记录生活、共同叙事。

核心体验指向一条原则：**让角色像"活人"一样存在**——有记忆、有主动行为、有独立社交关系、有可被翻看的生活痕迹。

### 1.3 本次迭代目标
- 将单文件改造为**模块化项目结构**，建立统一状态管理
- 上线**多角色系统**，支持自定义性格、头像、关系
- 上线**长期记忆系统**，AI 能跨对话记住用户信息
- 上线**角色主动行动线**，AI 具备主动发起对话的能力
- 上线**一起听歌**，与角色共享音乐时光
- **强化 Secret 功能**，可翻查角色手机中的各类生活记录
- **全局隐私开关**，控制角色间信息互通
- **朋友圈 AI 吃醋互动**，基于角色性格自动生成反应
- 通过前端 Notification API 实现**App 内消息弹窗**

### 3.2 👥 多角色系统（P0）

#### 概述
支持用户创建并切换多个 AI 角色，每个角色拥有独立形象、人格、记忆和状态。

#### 3.2.1 角色列表页
- 卡片式展示所有角色
- 每个角色显示：头像、名字、性格标签、状态
- 新增 / 编辑 / 删除
- 默认预置「骆云影」

#### 3.2.2 角色创建/编辑表单

| 字段 | 说明 |
|------|------|
| **头像** | 弹出选择面板：表情类（😎🥰🤨）/ 动物类（🐱🦊🐰）/ 符号类（⚡🌙🔥）/ 📷上传图片 |
| **名字** | 角色名称 |
| **性格描述** | 自由文本，AI 系统提示词核心 |
| **背景故事** | 角色身世、经历 |
| **与我的关系** | 下拉+自定义：恋人/朋友/对手/陌生人/师生/家人/自定义 |
| **说话风格** | 话多/话少、毒舌/温柔、爱用表情/不爱用、口头禅等 |
| **主动行动开关** | 是否允许该角色主动发消息 |
| **初始状态** | 在线/忙碌/离线 |

#### 3.2.3 角色状态

| 状态 | 表现 |
|------|------|
| 🟢 在线 | 秒回 |
| 🟡 忙碌 | 延迟回复，语气简洁 |
| ⚪ 离线 | 显示「最后上线」，消息暂存 |
| 🔴 勿扰 | 不接收消息，不弹通知 |

#### 3.2.4 角色切换
- 聊天页顶部角色切换器
- 切换后上下文/记忆/状态自动切换
- 不同角色聊天记录独立存储

### 3.4 🧠 长期记忆系统（P1 · 本次核心）

#### 3.4.1 记忆类型

| 类型 | 说明 | 有效期 |
|------|------|--------|
| 用户画像 | 人设固定信息 | 永久 |
| 重要事件 | 「上周感冒」「明天考试」 | 带时间戳，逐步淡化 |
| 偏好习惯 | 「爱喝咖啡」「怕黑」「喜欢XX歌」 | 永久 |
| 跨角色活动 | 「和角色B一起听了歌」 | 受隐私开关控制 |
| 共同经历 | 两人之间聊过的特定内容 | 角色私有，不互通 |
| 情绪状态 | 「今天不开心」 | 单次对话淡化 |

#### 3.4.2 跨角色互通逻辑

```
默认（隐私关）：所有角色共享信息池
  你=和B听了歌 → A知道 → A可能会吃醋
                  C知道 → 不关C的事就安静

隐私开：该活动不写入共享信息池
  你=和B听了歌（隐私开）→ A/C不知情
  但若你主动在A面前提起 → A知道了，可以反应
```

#### 3.4.3 记忆管理界面
- 角色详情页 → 「记忆库」面板
- 按时间线展示
- 手动新增/编辑/删除/标记重要/遗忘

### 3.5 🔄 角色主动行动线 + 🛡️ 全局隐私开关（P1 · 本次核心）

#### 3.5.1 触发场景

| 触发条件 | 行为 |
|----------|------|
| 时间触发 | 07:30 → 「醒了没」 |
| 长时间未对话（>6h） | 「一天没理我」 |
| 纪念日前夕 | 「明天是xx天」 |
| 天气变化 | 下雨 → 「带伞了吗」 |
| 记忆关联 | 记得你感冒 → 「好点没」 |
| **跨角色吃醋触发** | 你刚和A聊完/B聊完 → **隐私关时B/A主动发"你陪他不陪我？"** |
| 随机 | 日常闲聊 |

#### 3.5.2 🛡️ 全局隐私开关

| 位置 | 系统设置页 |
|------|-----------|
| **默认状态** | 关（信息互通） |
| **效果（关）** | 所有角色都知道你跟谁干了什么 |
| **效果（开）** | 当前活动仅参与者知道，其他角色不知情 |
| **吃醋触发条件** | 隐私=关 + 检测到你与角色A互动 → 角色B主动提起 |

#### 3.5.3 吃醋风格
- **定位**：开玩笑 / 撒娇，非真生气
- **骆云影版**："哦，跟他听歌开心是吧，那我去训练了"（等你来哄）
- **温柔版**："下次也可以找我呀"
- **频率**：偶尔，不是每次都说

#### 3.5.4 推送实现

纯前端 Notification API：
```
页面打开（可后台）
  → 定时器每5分钟检查
  → 条件匹配 → 调 API 生成消息
  → 弹系统通知 + 更新界面
```

### 3.6 🎧 一起听歌（P1 · 本次核心）

#### 入口
- 首页图标网格 → 🎵 一起听歌（独立入口，非聊天页附属）

#### 点歌方式
- **粘贴网易云链接**：自动识别格式，提取 song_id，加载播放
  - 支持 `https://music.163.com/song/<id>`
  - 支持 `https://y.music.163.com/m/song/<id>`
  - 支持短链接
- 支持推荐/随机

#### 界面
```
┌─────────────────────┐
│ 🎵 一起听歌    [✕]   │
│ ┌───────────────┐   │
│ │   专辑封面      │   │
│ │                │   │
│ └───────────────┘   │
│ 歌名 - 歌手         │
│ ═══●══════════      │
│ ◁◁  ⏸  ▷▷          │
│                     │
│ XX正在听... 🎧      │  ← 当前所选角色
│ ───────────────     │
│ "还行"              │  ← 50%概率出现
│                     │
│ [粘贴链接...]  🎵   │
└─────────────────────┘
```

#### AI 评论规则
- **切歌后 50% 概率**发一句评论，50%安静
- 评论内容按角色性格生成
- 一首歌不重复评论（除非重听）

#### 角色选择
- 进入时选择当前陪伴角色
- 可在听歌界面切换角色

#### 退出
- 点✕回到主页，音乐暂停
- 回到聊天页 AI 记得听过什么

### 3.7 🔍 悄悄话（翻AI手机）（P1 · 大幅增强）

#### 概述
用户可"偷看"角色的手机，查看角色的各种生活痕迹。所有内容由 AI 按角色性格生成。**同一个功能对不同角色展示不同内容**。

#### 主界面
模拟手机桌面，展示可点开的 App 图标：

| App | 内容 | 说明 |
|-----|------|------|
| 💬 **聊天记录** | 角色和其他人的对话（含和你的对话记录） | 已有，支持多角色切换 |
| 📓 **记事本** | 角色写的日记/随笔 | 已有，AI 生成 |
| 🎵 **音乐软件** | 角色最近听的歌、收藏歌单、听歌报告 | 新增，与一起听歌联动 |
| 🖼 **相册** | 角色手机里的照片（概念图/AI描述） | 已有 |
| 🍔 **外卖订单** | 角色最近点了什么外卖、常去哪家店 | 新增 |
| 🌐 **浏览器记录** | 角色最近搜索了什么、浏览了什么 | 新增 |
| ✉️ **匿名信箱** | 用户给角色写的信 + 角色回信 | 新增 |

#### 3.7.1 外卖订单
- AI 生成角色近期外卖记录（符合性格）
- 骆云影版："深夜点了炸鸡""咖啡月卡"
- 可点击查看订单详情（AI 生成）

#### 3.7.2 音乐软件
- 角色最近播放歌单
- 收藏/喜欢的歌
- 和一起听歌联动：一起听过的歌会出现在"最近播放"里

#### 3.7.3 浏览器搜索记录
- AI 生成角色在搜索框里输入过的内容
- **最有叙事潜力的部分**，示例：
  - "怎么哄生气的女朋友"
  - "xx（用户名字）喜欢什么"
  - "傲娇的人怎么表达关心"
  - "今天天气"
- 按角色性格决定画风

#### 3.7.4 匿名信箱
- 用户可写匿名信给角色
- 角色回信（AI 生成）

### 3.8 🌄 朋友圈 + AI 自动互动（P2）

#### 3.8.1 基础
- 发动态
- 角色自动点赞/评论

#### 3.8.2 吃醋机制
- AI 扫描朋友圈文字，识别是否出现其他角色名字
- 匹配角色列表中的名字/昵称
- 若匹配到 → 按性格生成吃醋言论
- 吃醋风格：撒娇/开玩笑级别

#### 3.8.3 可见范围
- 仅自己 / 角色可见 / 所有人可见

### 3.9 ⏱ 陪伴计时 / 任务 / 闹钟 / 记账 / 潮汐

```
你学了一小时
→ 骆云影："学完了？一小时够拼的"
→ 温柔角色B："好认真，要休息一下吗"
```

**记账示例**：
```
今天花了200
→ 骆云影："又买什么了"
→ 节俭角色C："这个月花了不少啊"
```
