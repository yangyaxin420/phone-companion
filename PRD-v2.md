## 二、详细改动

### 2.1 项目结构拆分为模块

**之前**：单文件 `index.html`（约203KB，CSS+HTML+JS全部内联）

**之后**：
```
phone-companion/
├── index.html           # ~975行，外链 CSS + JS
├── css/
│   └── style.css        # 所有样式
├── js/
│   ├── utils.js         # 工具函数
│   ├── app.js           # 核心（导航/状态/时钟/图标/倒数日）
│   ├── weather.js       # 天气 API（3种方案）
│   ├── tasks.js         # 日程 + 闹钟
│   ├── expense.js       # 记账（含图表 + 导入导出清空）
│   ├── chat.js          # 聊天AI + 表情 + 多角色数据
│   ├── secret.js        # Secret翻手机（多角色版）
│   ├── moments.js       # 朋友圈
│   ├── companion.js     # 专注伴伴增强版
│   ├── persona.js       # 用户人设 + API配置 + 数据管理
│   ├── tide.js          # 潮汐 + 每日心情
│   └── settings.js      # 设置（预留）
├── manifest.json / sw.js
```

### 2.2 主页 UI 改版（灰调ins风）

- **壁纸底纹**：几何线条 + 渐变背景
- **问候语**：「你好，晞晞 · 今天想做点什么？」+ 当前日期
- **天气卡片**：白底圆角，字体缩小（温度32→24）
- **倒数日卡片**：白底，字体缩小（天数36→28）
- **图标网格**：白底圆角图标 + 灰色名称文字（`app-grid`）
- **底部导航**：增加文字标签（返回/主页/下一屏）
- **删除图标**：日程、闹钟
- **保留图标**：聊天、专注、朋友圈、Secret、记账、潮汐、人设

### 2.3 专注伴伴增强

- **任务输入**：专注前填写任务名
- **时长设置**：+/- 按钮调整分钟（正计时/倒计时）
- **模式切换**：⌛ 正计时 / ⏳ 倒计时
- **铃声**：上传/试听/重置自定义铃声
- **全屏计时界面**：深色背景（`#1a1a2e`），大字显示，专注中自动进入
- **AI气泡鼓励**：专注中 AI 按角色性格弹出鼓励/吐槽
- **专注记录**：每次完成自动保存，本周/本月/全部统计

### 2.4 多角色聊天列表

- **数据改造**：`chatMessages` 单数组 → `chatData = { charId: [messages] }`
- **对话列表页**：微信风格，显示头像/名字/状态/最后消息/时间
- **按时间排序**：最新消息的角色排最前
- **角色切换**：点击角色进入聊天，数据/人设/提示词自动切换
- **旧数据兼容**：旧聊天记录自动归入骆云影

### 2.5 角色管理

- **添加角色**：对话列表右上角「+ 角色」，输入名字即可创建
- **删除角色**：点击「···」进入管理模式 → 角色旁出现 ✕ → 确认删除
- **保护逻辑**：至少保留一个角色，删除当前角色自动切到第一个

### 2.6 Secret 多角色翻手机

- **角色切换器**：Secret 桌面顶部显示所有角色按钮，点击切换
- **数据隔离**：每个角色的 Secret 内容独立存储
- **AI间聊天记录**：联系人生成中包含其他AI角色，可查看对话
- **离线降级**：无 API Key 时根据性格生成不同内容（傲娇/温柔/高冷三种模板）
- **有 API 时**：按角色性格生成更丰富的内容

### 2.7 记账

- **补回输入表单**：金额/分类/备注/日期
- **导出**：导出为 JSON 文件
- **导入**：导入 JSON 文件，与现有数据合并
- **清空**：双重确认后清空全部记账数据
- **自动渲染**：导航到记账页时自动渲染分类按钮

### 2.8 用户人设系统

**页面**：点「人设」图标 → 「🧍 我的人设」

| 字段 | 默认值 |
|------|--------|
| 我的名字 | 晞晞 |
| 性别 | 女 |
| 年龄/身份 | 大一，社会工作 |
| 性格标签 | 敏感、爱写东西、嘴硬心软 |
| 兴趣爱好 | 写作、听歌、骆云影 |
| 重要背景 | 在杭州读书 |

**AI 上下文注入**：用户人设自动注入 AI 系统提示，让 AI 知道在和谁说话

### 2.9 AI 人设迁移

- **从人设页移除**：AI 名称、背景故事、世界书、关系不再在「人设」页
- **移到聊天页**：右上角 ⚙ 弹窗，可编辑：
  - AI 名字
  - 背景故事
  - 说话风格（System Prompt）
  - 世界书
  - 与我的关系（下拉选择）

### 2.10 关系系统

- **每个角色独立关系**：恋人/朋友/对手/陌生人/师生/家人/自定义
- **在 ⚙ 弹窗设置**：切换角色后关系自动切换
- **AI 知道关系**：上下文注入当前角色与用户的关系

### 2.11 聊天页飞鸟壁纸

- **底色**：`#E2E8F0` 浅灰蓝 + 蓝白渐变
- **飞鸟底纹**：6只浅灰蓝色飞鸟 SVG 散布页面
- **气泡配色**：
  - AI 气泡：白色底 + 阴影 + 深灰文字
  - 用户气泡：石板灰 `#475569` + 白色文字
- **输入栏/头部**：半透明毛玻璃效果

### 2.12 PWA 安装修复

- **添加 manifest 链接**：HTML 头部引入 `manifest.json`
- **完善 sw.js**：添加 `fetch` 事件处理器，支持缓存
- **manifest 配置**：`display: standalone`、`scope`、图标 `purpose`
- **主题色**：更新为 `#F8FAFC`
- **缓存刷新**：CSS 添加 `?v=2` 参数

### 2.13 API 配置简化

- **删除多余字段**：移除 baseUrl、model、CORS 代理等复杂选项
- **仅保留 Key 输入**：配置页只留 API Key 输入框
- **默认值**：baseUrl 默认 `https://api.deepseek.com`，model 默认 `deepseek-chat`

## 三、页面结构

```
主页 → 问候 + 天气 + 倒数日 + 7个应用图标
│
├── 💬 聊天列表 → 所有角色对话列表 → 点角色进入聊天
│   └── 聊天页（⚙ AI人设/关系编辑）
├── ⟳ 专注 → 任务/时长/模式/铃声 → 开始→全屏计时（AI气泡）
├── ○ 朋友圈 → 发布动态、AI评论
├── ◎ Secret → 角色切换器 → 聊天记录/记事本/歌单/相册
├── ￥ 记账 → 表单输入 + 列表 + 统计图表（导入/导出/清空）
├── ☽ 潮汐 → 月经日历 + 预测 + 心情记录
└── ≡ 人设 → 用户人设 + API配置 + 数据管理
```
