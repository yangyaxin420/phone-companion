## 二、详细改动

### 2.1 项目结构拆分为模块

**之前**：单文件 `index.html`（约203KB，CSS+HTML+JS全部内联）

**之后**：
```
phone-companion/
├── index.html           # ~950行，外链 CSS + JS
├── css/
│   └── style.css        # 所有样式
├── js/
│   ├── utils.js         # 工具函数
│   ├── app.js           # 核心（导航/状态/时钟/图标/倒数日）
│   ├── weather.js       # 天气 API（3种方案）
│   ├── tasks.js         # 日程 + 闹钟
│   ├── expense.js       # 记账（含图表）
│   ├── chat.js          # 聊天AI + 表情 + 多角色数据
│   ├── secret.js        # Secret翻手机（多角色版）
│   ├── moments.js       # 朋友圈
│   ├── companion.js     # 专注伴伴增强版
│   ├── persona.js       # 用户人设 + API配置 + 数据管理
│   ├── tide.js          # 潮汐 + 每日心情
│   └── settings.js      # 设置（预留）
├── manifest.json / sw.js
```

### 2.2 主页 UI 改版（灰调ins风）

- **壁纸底纹**：几何线条 + 渐变背景
- **问候语**：「你好，晞晞 · 今天想做点什么？」+ 当前日期
- **天气卡片**：白底圆角，字体缩小（温度32→24）
- **倒数日卡片**：白底，字体缩小（天数36→28）
- **图标网格**：白底圆角图标 + 灰色名称文字
- **底部导航**：增加文字标签（返回/主页/下一屏）
- **删除图标**：日程、闹钟
- **保留图标**：聊天、专注、朋友圈、Secret、记账、潮汐、人设

### 2.3 专注伴伴增强

- **任务输入**：专注前填写任务名
- **时长设置**：+/- 按钮调整分钟
- **模式切换**：正计时 / 倒计时
- **铃声**：上传/试听/重置自定义铃声
- **全屏计时界面**：深色背景，大字显示，专注中自动进入
- **AI气泡鼓励**：专注中 AI 按角色性格弹出鼓励/吐槽
- **专注记录**：每次完成自动保存，按周/月/全部统计

### 2.4 多角色聊天列表

- **数据改造**：`chatMessages` 单数组 → `chatData = { charId: [messages] }`
- **对话列表页**：微信风格，显示头像/名字/状态/最后消息/时间
- **按时间排序**：最新消息的角色排最前
- **角色切换**：点击角色进入该角色的聊天（数据/人设/提示词自动切换）
- **旧数据兼容**：旧聊天记录自动归入骆云影

### 2.5 角色管理

- **添加角色**：对话列表右上角「+ 角色」，输入名字即可创建
- **删除角色**：点击「···」进入管理模式，角色旁出现红色 ✕，点击确认删除
- **保护逻辑**：至少保留一个角色，删除当前角色自动切到第一个

### 2.6 Secret 多角色翻手机

- **角色切换器**：Secret 桌面顶部显示所有角色按钮，点击切换
- **数据隔离**：每个角色的 Secret 内容独立存储
- **AI间聊天记录**：生成的联系人中包含其他AI角色，可查看对话
- **离线降级**：无 API Key 时根据性格生成不同内容（傲娇/温柔/高冷三种模板）
- **有 API 时**：按角色性格生成更丰富的内容

### 2.7 记账表单修复

- 补回金额/分类/备注/日期输入字段
- 导航到记账页时自动渲染分类按钮

### 2.8 用户人设系统

**页面**：点「人设」图标 → 「🧍 我的人设」

| 字段 | 默认值 |
|------|--------|
| 我的名字 | 晞晞 |
| 性别 | 女 |
| 年龄/身份 | 大一，社会工作 |
| 性格标签 | 敏感、爱写东西、嘴硬心软 |
| 兴趣爱好 | 写作、听歌、骆云影 |
| 重要背景 | 在杭州读书 |

**AI 上下文注入**：用户人设信息自动注入 AI 系统提示，让 AI 知道在和谁说话

### 2.9 AI 人设迁移

- **从人设页移除**：AI 名称、背景故事、世界书不再在「人设」页
- **移到聊天页**：聊天页右上角 ⚙ 弹窗，可编辑：
  - AI 名字
  - 背景故事
  - 说话风格（System Prompt）
  - 世界书
  - 与我的关系（下拉选择）

### 2.10 关系系统

- **每个角色独立关系**：恋人/朋友/对手/陌生人/师生/家人/自定义
- **在 ⚙ 弹窗设置**：切换角色后关系自动切换
- **AI 知道关系**：上下文注入当前角色与用户的关系

## 三、页面结构

```
主页 → 问候 + 天气 + 倒数日 + 7个应用图标
│
├── 💬 聊天列表 → 所有角色对话列表 → 点角色进入聊天

├── ⟳ 专注 → 任务/时长/模式/铃声 → 开始→全屏计时（AI气泡）
├── ○ 朋友圈 → 发布动态、AI评论
├── ◎ Secret → 角色切换器 → 聊天记录/记事本/歌单/相册
├── ￥ 记账 → 表单输入 + 列表 + 统计图表
├── ☽ 潮汐 → 月经日历 + 预测 + 心情记录
└── ≡ 人设 → 用户人设 + API配置 + 数据管理
```
